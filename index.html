Replacr

String Replacement Utility for Windows

Replacr is a simple app that allows you to replace a pattern in a string and get the result in an output field. It is a convenient utility when large amounts of repetitive replacement is needed and automation is not an option.

Replacement patterns can be created, stored and reused.

This application was created primarily for educational purposes.
